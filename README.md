# 🏘️ Smart Community Web System

A web-based community management system built as part of an Open Ended Lab (OEL) project. It enables residents and admins to interact with a centralized platform for community-related activities.

## 🌐 Live Pages

| Page | Description |
|------|-------------|
| `index.html` | Dashboard — overview & quick stats |
| `login.html` | Login with role selection |
| `register.html` | New resident registration |
| `announcements.html` | Community announcements board |
| `complaints.html` | Submit complaints or requests |
| `residents.html` | Residents directory (admin view) |

---

## ✨ Features

- 🔐 Role-based access (Admin / Resident) using localStorage
- 📋 Announcements board with dynamic display
- 📝 Complaint submission form with validation
- 👥 Residents directory with search/filter
- 📱 Fully responsive design using Bootstrap 5
- ✅ JS form validation on all input forms
- 🚫 Route protection (redirects if not logged in)

## 🚀 How to Run

1. Clone the repository:
   ```
   git clone https://github.com/Aalishbaali-77/SmartCommunity.git
   ```
2. Open the folder in **VS Code**
3. Right-click `login.html` → **Open with Live Server**
4. Login using any email + password (min 6 chars) + select a role
