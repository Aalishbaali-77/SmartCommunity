## 🌐 Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page — hero, news ticker, notices, quick access |
| `login.html` | Login with role selection (Admin / Resident) |
| `register.html` | New resident registration with validation |
| `dashboard.html` | Personalised dashboard with Chart.js analytics |
| `notices.html` | Community notice board with search & filter |
| `request.html` | Submit & track service requests |
| `admin.html` | Admin panel — manage requests, residents, notices & analytics |
| `profile.html` | View & edit profile, change password, activity log |

---

## ✨ Features

- 🔐 Role-based access control (Admin / Resident) using localStorage
- 📋 Community notice board with search, filter, post & delete
- 🛠️ Service request form with category, priority & status tracking
- 📊 Chart.js dashboard — doughnut & bar charts
- 👥 Admin panel with full resident & request management
- ✅ JS form validation on all input forms
- 🔔 Toast notifications on all actions
- 📰 Live news ticker on home page
- 🌙 Dark mode toggle — persists across all pages
- 📱 Fully responsive design using Bootstrap 5
- 🚫 Auth guard — redirects to login if not logged in

## 🚀 How to Run

1. Clone the repository:
git clone https://github.com/Aalishbaali-77/SmartCommunity.git
2. Open the folder in **VS Code**
3. Install **Live Server** extension by Ritwick Dey
4. Right-click `login.html` → **Open with Live Server**
5. Login using the demo credentials above

> No installation or build process needed — runs directly in the browser!
